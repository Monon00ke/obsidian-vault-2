# Задание 1а (исправленный вариант)  
Исследовать сходимость ряда  
$\sum\limits_{n=1}^{\infty} \sqrt{ \dfrac{n+2}{n+1} }$  
с помощью **необходимого признака сходимости**  

**Уточнение от пользователя:**  
Корень квадратный стоит над всей дробью $\dfrac{n+2}{n+1}$, а не только над числителем.  
(Оригинальная интерпретация была неверной из-за почерка на фото.)

## Необходимый признак (признак расходимости)

> Если ряд $\sum a_n$ **сходится**, то **обязательно**  
> $\displaystyle\lim_{n\to\infty} a_n = 0$  

**Зачем нужен этот признак?**  
- Это **самый простой** тест на расходимость.  
- Если lim a_n ≠ 0 (или предел не существует), ряд **точно расходится** — и исследование закончено.  
- Если lim a_n = 0, признак **не даёт информации** о расходимости (нужно другие тесты).  

## Решение шаг за шагом  

### Шаг 1: Записываем общий член ряда  
$$
a_n = \sqrt{ \frac{n+2}{n+1} }  
$$  

**Зачем?**  
- Чтобы четко работать с выражением для предела.  
- Избегаем ошибок интерпретации (как с фото).  

### Шаг 2: Упрощаем выражение a_n для анализа при n → ∞  
$$
a_n = \sqrt{ \frac{n+2}{n+1} } = \sqrt{ \frac{ (n+1) + 1 }{n+1} } = \sqrt{ 1 + \frac{1}{n+1} }  
$$  

**Зачем?**  
- Упрощение показывает асимптотику: при больших n дробь → 1 + 0 = 1.  
- Легче вычислить предел без лишних вычислений.  

### Шаг 3: Находим предел lim_{n→∞} a_n  
$$
\lim_{n\to\infty} a_n = \lim_{n\to\infty} \sqrt{ 1 + \frac{1}{n+1} } = \sqrt{ 1 + 0 } = \sqrt{1} = 1  
$$  

**Зачем?**  
- Это ключевое действие признака: сравниваем с 0.  
- Если ≠ 0 → расходимость доказана сразу.  

### Шаг 4: Применяем необходимый признак  
- lim a_n = 1 ≠ 0.  
- Следовательно, необходимое условие **не выполнено**.  

**Зачем этот вывод?**  
- Прямо следует из формулировки: если lim ≠ 0, ряд не может сходиться.  

## Вывод  

Ряд **расходится** по необходимому признаку сходимости (предел общего члена равен 1, а не 0).  

## Краткий итог (для зачёта / ответа)  
- lim a_n = 1 ≠ 0.  
- **Расходится** (необходимый признак).  

## Что дальше? (для информации)  
- Другие признаки здесь не нужны, т.к. расходимость уже доказана.  
- Интуитивно: члены a_n ≈ 1 при больших n, сумма бесконечных "1" расходится.
