# 📐 Шпаргалка: Производные и интегралы

---

## 1. Основные производные

| Функция | Производная |
|---------|-------------|
| $x^n$ | $n x^{n-1}$ |
| $e^x$ | $e^x$ |
| $a^x$ | $a^x \ln a$ |
| $\ln x$ | $\frac{1}{x}$ |
| $\log_a x$ | $\frac{1}{x \ln a}$ |
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $\tan x$ | $\frac{1}{\cos^2 x}$ |
| $\cot x$ | $-\frac{1}{\sin^2 x}$ |
| $\arcsin x$ | $\frac{1}{\sqrt{1-x^2}}$ |
| $\arccos x$ | $-\frac{1}{\sqrt{1-x^2}}$ |
| $\arctan x$ | $\frac{1}{1+x^2}$ |
| $\text{arc cot } x$ | $-\frac{1}{1+x^2}$ |

**Производная сложной функции:**
$$
(f(g(x)))' = f'(g(x)) \cdot g'(x)
$$

---

## 2. Основные интегралы

| Функция | Интеграл |
|---------|----------|
| $x^n$ | $\frac{x^{n+1}}{n+1} + C$ ($n \neq -1$) |
| $\frac{1}{x}$ | $\ln \|x\| + C$ |
| $e^x$ | $e^x + C$ |
| $a^x$ | $\frac{a^x}{\ln a} + C$ |
| $\sin x$ | $-\cos x + C$ |
| $\cos x$ | $\sin x + C$ |
| $\frac{1}{\cos^2 x}$ | $\tan x + C$ |
| $\frac{1}{\sin^2 x}$ | $-\cot x + C$ |
| $\frac{1}{\sqrt{1-x^2}}$ | $\arcsin x + C$ |
| $\frac{1}{1+x^2}$ | $\arctan x + C$ |

---

## 3. Интегрирование по частям

**Формула:**
$$
\int u\, dv = uv - \int v\, du
$$

**Как выбирать $u$ и $dv$:**

- $u$ — то, что упрощается при дифференцировании (логарифм, обратная тригонометрия, полином)
- $dv$ — то, что легко интегрируется (экспонента, синус, косинус)

**Примеры:**
- $\int x e^x dx$ → $u = x$, $dv = e^x dx$
- $\int \ln x\, dx$ → $u = \ln x$, $dv = dx$
- $\int x^2 \sin x\, dx$ → $u = x^2$, $dv = \sin x\, dx$

---

## 4. Замена переменной

**Формула:**
$$
\int f(g(x)) g'(x)\, dx = \int f(t)\, dt \quad (t = g(x))
$$

**Когда применять:**
- Под корнем или в знаменателе есть выражение
- Есть линейная функция внутри сложной

**Примеры:**
- $\int \frac{dx}{\sqrt{2x+3}}$ → $t = 2x+3$
- $\int e^{5x} dx$ → $t = 5x$
- $\int \sin(3x) dx$ → $t = 3x$

---

## 5. Интегралы от тригонометрических функций

### $\int \sin^m x \cos^n x\, dx$

- Если $m$ нечётное → выделяем $\sin x$ и делаем замену $t = \cos x$
- Если $n$ нечётное → выделяем $\cos x$ и делаем замену $t = \sin x$
- Если оба чётные → используем формулы понижения степени

### $\int \sin ax \cos bx\, dx$

Используем формулы произведения:
$$
\sin A \cos B = \frac{1}{2} [\sin(A+B) + \sin(A-B)]
$$

### $\int \tan x\, dx$
$$
\int \tan x\, dx = -\ln |\cos x| + C
$$

---

## 6. Рациональные дроби (разложение на простейшие)

**Общий план:**

1. Если степень числителя ≥ степени знаменателя → делим многочлены
2. Разлагаем знаменатель на множители
3. Пишем разложение:

| Вид знаменателя | Разложение |
|------------------|------------|
| $(x-a)$ | $\frac{A}{x-a}$ |
| $(x-a)^k$ | $\frac{A_1}{x-a} + \frac{A_2}{(x-a)^2} + \dots + \frac{A_k}{(x-a)^k}$ |
| $x^2 + px + q$ (неразложимый) | $\frac{Ax + B}{x^2 + px + q}$ |

**Пример:**
$$
\frac{3x+2}{(x-1)(x+2)} = \frac{A}{x-1} + \frac{B}{x+2}
$$

Находим $A$ и $B$ методом неопределённых коэффициентов.

## 7. Полезные тригонометрические тождества

$$
\sin^2 x + \cos^2 x = 1
$$

$$
\sin 2x = 2\sin x \cos x
$$

$$
\cos 2x = \cos^2 x - \sin^2 x = 2\cos^2 x - 1 = 1 - 2\sin^2 x
$$

$$
\tan 2x = \frac{2\tan x}{1 - \tan^2 x}
$$

$$
1 + \tan^2 x = \frac{1}{\cos^2 x}
$$

---

## 8. Полезные формулы

**Производная произведения:**
$$
(uv)' = u'v + uv'
$$

**Производная частного:**
$$
\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}
$$

**Интеграл по частям (ещё раз):**
$$
\int u dv = uv - \int v du
$$

**Замена (ещё раз):**
$$
\int f(g(x)) g'(x) dx = \int f(t) dt
$$
