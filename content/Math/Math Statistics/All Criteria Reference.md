# 📊 Все критерии матстата — пошаговое решение

Сначала — термины, чтобы понимать, о чём речь дальше.

## Словарик

| Слово | Смысл |
|-------|-------|
| **$H_0$ (нулевая гипотеза)** | «Ничего не изменилось». Мы пытаемся **опровергнуть** её |
| **$H_1$ (альтернатива)** | «Что-то изменилось». Что доказываем |
| **$\alpha$** | Как часто мы готовы ошибиться. Обычно **0.05** = 1 раз из 20 |
| **p-value** | «Вероятность, что мы ошиблись». Если **p < α** → $H_0$ отвергаем |
| **Статистика критерия** | Число, которое считаем по данным и формуле |
| **Критическое значение** | Граница «отсечки» из таблицы |
| **Степень свободы (df)** | Сколько «независимых кусочков информации» осталось |
| **Стандартное отклонение ($\sigma$, $s$)** | «Средняя разбросанность» данных от среднего |
| **Дисперсия ($\sigma^2$, $s^2$)** | Стандартное отклонение в квадрате |
| **Выборка** | Несколько объектов, которые мы измерили |
| **Объём выборки ($n$)** | Сколько объектов измерили |

## 1. z-критерий (среднее vs число, σ известна)

**Когда:** знаем $\sigma$, проверяем, равно ли среднее какой-то константе

**Пример:** В популяции IQ распределён со средним 100 и σ = 15. Мы измерили IQ у 50 своих друзей, получили среднее 105. Друзья умнее среднего?

**Пошаговый ход решения:**

| Шаг | Что делаем | Формула / Действие | Результат |
|:---:|------------|--------------------|:---------:|
| 1 | Формулируем $H_0$ и $H_1$ | $H_0: \mu = 100$ (друзья не отличаются) | — |
| | | $H_1: \mu \neq 100$ (отличаются) | |
| 2 | Выбираем $\alpha$ | Обычно 0.05 | **0.05** |
| 3 | Собираем данные | $n=50$, $\bar{x}=105$, $\sigma=15$ | — |
| 4 | Считаем **статистику z** | $z = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}} = \frac{105 - 100}{15 / \sqrt{50}}$ | **2.36** |
| 5 | Ищем критическое значение | Для $\alpha=0.05$ двусторонний → $z_{кр} = 1.96$ | **1.96** |
| 6 | **Сравниваем** | $|2.36| > 1.96$? Да ✓ | — |
| 7 | **Вывод** | Отвергаем $H_0$ → Друзья умнее среднего | — |

**Откуда 1.96?** Это такое число, что 95% всех значений нормального распределения лежат в пределах $(-1.96; +1.96)$. Если наша статистика вылезла за эти границы — это редкое событие (вероятность < 5%). Значит, $H_0$ неправильна.

**p-value:** $p = 2 \cdot (1 - \Phi(2.36)) \approx 0.018$. $0.018 < 0.05$ → отвергаем. Смысл: «если бы $H_0$ была верна, мы бы увидели такое редкое событие с вероятностью 1.8% — слишком маловероятно».

**Python:**
```python
from scipy import stats
z = (105 - 100) / (15 / 50**0.5)          # 2.36
p = 2 * (1 - stats.norm.cdf(abs(z)))      # 0.018
```

## 2. t-критерий (среднее vs число, σ неизвестна)

**Когда:** σ **НЕ** знаем (почти всегда), оцениваем по $s$

**Пример:** Поставщик говорит «моя пшеница даёт 60 ц/га». Мы проверили 37 полей, получили среднее 55, стандартное отклонение 3. Врёт?

| Шаг | Что делаем | Действие | Рез. |
|:---:|------------|----------|:----:|
| 1 | $H_0$ | $\mu = 60$ | — |
| | $H_1$ | $\mu < 60$ (левосторонняя) | |
| 2 | $\alpha$ | 0.05 | |
| 3 | Данные | $n=37$, $\bar{x}=55$, $s=3$ | |
| 4 | **Статистика t** | $t = \frac{55 - 60}{3 / \sqrt{37}} = \frac{-5}{0.493}$ | **-10.14** |
| 5 | **Степени свободы** | $df = n - 1 = 36$ | **36** |
| 6 | Крит. значение | $t_{0.05; 36}$ (левостор.) из таблицы Стьюдента | **-1.69** |
| 7 | **Сравниваем** | $-10.14 < -1.69$? Да ✓ | |
| 8 | **Вывод** | Отвергаем $H_0$ → поставщик врёт | |

**Откуда $t$?** Мы не знаем $\sigma$, поэтому вместо неё используем $s$ (оценку по выборке). Но $s$ — неточная, поэтому распределение чуть шире нормального, особенно при малых $n$. Это распределение **Стьюдента** (t-распределение).

**Откуда -1.69?** Это 5%-я граница слева для t-распределения с 36 степенями свободы. Потому что альтернатива левосторонняя ($\mu < 60$), нас интересует только левый хвост.

**Почему df = 36?** Потому что мы оценили одно число (среднее) по data. Осталось $n - 1 = 36$ «свободных» наблюдений.

**Python:**
```python
t_stat, p = stats.ttest_1samp(data, 60, alternative='less')
# t_stat ≈ -10.14, p ≈ 0.0000002 < 0.05 → отвергаем
```

## 3. F-критерий (сравнение дисперсий)

**Когда:** проверить, одинакова ли разбросанность (дисперсия) двух групп

**Пример:** Два станка делают болты. Первый: $s_1 = 4$ мм, 15 болтов. Второй: $s_2 = 7$ мм, 15 болтов. Одинакова ли точность?

| Шаг | Что делаем | Действие | Рез. |
|:---:|------------|----------|:----:|
| 1 | $H_0$ | $\sigma_1^2 = \sigma_2^2$ (точность одинакова) | |
| | $H_1$ | $\sigma_2^2 > \sigma_1^2$ (второй менее точен) | |
| 2 | $\alpha$ | 0.01 | |
| 3 | Данные | $s_1=4$, $n_1=15$; $s_2=7$, $n_2=15$ | |
| 4 | **Статистика F** | Делим **большую** дисперсию на меньшую: $F = \frac{7^2}{4^2} = \frac{49}{16}$ | **3.0625** |
| 5 | Степени свободы | $df_1 = n_1 - 1 = 14$ (числитель) | **14** |
| | | $df_2 = n_2 - 1 = 14$ (знаменатель) | **14** |
| 6 | Крит. значение | $F_{0.01; 14, 14}$ из таблицы Фишера | **≈ 3.74** |
| 7 | **Сравниваем** | $3.0625 < 3.74$? Да (не превысило) | |
| 8 | **Вывод** | $H_0$ **не отвергаем** → не доказали, что точность разная | |

**Почему именно F?** Отношение двух дисперсий подчиняется распределению Фишера. Если дисперсии равны, F ≈ 1. Если F сильно больше 1 — дисперсии разные.

**Почему 3.74?** Это граница, выше которой F попадает только в 1% случаев, если дисперсии на самом деле равны. Наше F = 3.06 не дотянуло.

**Python:**
```python
F = 7**2 / 4**2
p = 1 - stats.f.cdf(F, 14, 14)  # ≈ 0.021
# 0.021 > 0.01 → не отвергаем
```

## 4. χ² Пирсона (критерий согласия)

**Когда:** проверить, подчиняются ли данные какому-то распределению (обычно нормальному)

**Пример:** 209 раз записали число покупок за день. Похоже ли на нормальное распределение?

**Логика:** если данные нормальны, мы знаем, сколько значений должно попасть в каждый интервал (по формуле нормального распределения). Сравниваем с реальностью.

| Шаг | Что делаем | Действие | Рез. |
|:---:|------------|----------|:----:|
| 1 | $H_0$ | Данные ~ нормальны | |
| | $H_1$ | Данные **не** нормальны | |
| 2 | $\alpha$ | 0.05 | |
| 3 | Разбиваем на k интервалов | $k=11$ интервалов | **11** |
| 4 | Считаем $O_i$ для каждого | Например, в первый интервал попало **7** значений | |
| 5 | Считаем $E_i$ для каждого | По формуле нормального распределения: $E_i = n \cdot P_i$ | |
| | | $P_i = \Phi(\frac{x_{верх} - \bar{x}}{s}) - \Phi(\frac{x_{низ} - \bar{x}}{s})$ | |
| 6 | **Статистика χ²** | $\chi^2 = \sum \frac{(O_i - E_i)^2}{E_i}$ | **≈ 21.8** |
| 7 | Степени свободы | $df = k - 1 - m$ | **8** |
| | | $m=2$ (оценили среднее и стд у нормального) | |
| 8 | Крит. значение | $\chi^2_{0.05; 8}$ из таблицы χ² | **≈ 15.5** |
| 9 | **Сравниваем** | $21.8 > 15.5$? Да ✓ | |
| 10 | **Вывод** | Отвергаем $H_0$ → распределение **не нормальное** | |

**Почему df = 8?** Было 11 интервалов ($k$). Минус 1 (так как сумма частот = n фиксирована). Минус 2 (оценили среднее и стд). Итого: $11 - 1 - 2 = 8$.

**Суть:** Если $\chi^2$ маленький — теория совпадает с реальностью. Если большой — расходится. 21.8 > 15.5 → расходится сильно.

**Python:**
```python
chi2, p = stats.chisquare(f_obs, f_exp)
```

## 5. χ² независимости (таблицы сопряжённости)

**Когда:** два категориальных признака — зависимы или нет?

**Пример:** 300 человек. Тип темперамента (холерик, сангвиник, флегматик) и профессия (врач, инженер, артист). Есть связь?

| Шаг | Что делаем | Действие | Рез. |
|:---:|------------|----------|:----:|
| 1 | $H_0$ | Темперамент и профессия **независимы** | |
| | $H_1$ | **Зависимы** | |
| 2 | $\alpha$ | 0.05 | |
| 3 | Строим таблицу $3 \times 3$ | В каждой клетке — сколько человек с таким сочетанием | |
| 4 | Считаем **ожидаемые** частоты | $E_{ij} = \frac{(\text{сумма строки}) \cdot (\text{сумма столбца})}{n}$ | |
| 5 | **Статистика χ²** | $\chi^2 = \sum \frac{(n_{ij} - E_{ij})^2}{E_{ij}}$ | **≈ 30** |
| 6 | Степени свободы | $df = (r-1)(c-1) = (3-1)(3-1)$ | **4** |
| 7 | Крит. значение | $\chi^2_{0.05; 4}$ из таблицы | **≈ 9.49** |
| 8 | **Сравниваем** | $30 > 9.49$? Да ✓ | |
| 9 | **Вывод** | Отвергаем $H_0$ → связь **есть** | |

**Логика:** Если признаки независимы, то частота в каждой клетке = (процент строки) × (процент столбца) × n. Если реальные частоты сильно отличаются от ожидаемых — признаки зависимы.

**Сила связи (V Крамера):**
$$V = \sqrt{\frac{30}{300 \cdot 2}} \approx 0.224$$
Умеренная связь.

**Python:**
```python
chi2, p, dof, expected = stats.chi2_contingency(table)
V = np.sqrt(chi2 / (n * min(r-1, c-1)))
```

## 6. ANOVA (3+ группы, сравнение средних)

**Когда:** надо сравнить средние трёх и более групп

**Пример:** Три метода обучения. Результаты: Метод 1 (ср. 4.0), Метод 2 (ср. 2.0), Метод 3 (ср. 5.0). Есть ли разница?

**Логика:** смотрим две суммы квадратов:
- **SSB** (между группами) — насколько средние групп отличаются от общего среднего
- **SSW** (внутри групп) — насколько люди внутри одной группы разбросаны

Если $SSB \gg SSW$ → группы реально разные.

| Шаг | Что делаем | Действие | Рез. |
|:---:|------------|----------|:----:|
| 1 | $H_0$ | $\mu_1 = \mu_2 = \mu_3$ (все методы одинаковы) | |
| | $H_1$ | Хотя бы один отличается | |
| 2 | $\alpha$ | 0.05 | |
| 3 | Считаем общее среднее | $\bar{x} = \frac{4+2+5}{3} = 3.67$ | **3.67** |
| 4 | **SSB** | $\sum n_i (\bar{x}_i - \bar{x})^2 = 3\cdot(4-3.67)^2 + 3\cdot(2-3.67)^2 + 3\cdot(5-3.67)^2$ | **13.5** |
| 5 | **SSW** | $\sum_{i,j} (x_{ij} - \bar{x}_i)^2$ (сумма разбросов внутри групп) | **6** |
| 6 | **F-статистика** | $F = \frac{SSB / (k-1)}{SSW / (n-k)} = \frac{13.5 / 2}{6 / 6}$ | **7.0** |
| 7 | Степени свободы | $df_1 = k-1 = 2$, $df_2 = n-k = 6$ | |
| 8 | p-value | По распределению Фишера | **≈ 0.027** |
| 9 | **Сравниваем** | $0.027 < 0.05$? Да ✓ | |
| 10 | **Вывод** | Отвергаем $H_0$ → методы дают разный результат | |

**Почему F?** Отношение «межгрупповой разброс / внутригрупповой разброс». Если методы одинаковы, F ≈ 1. Если F > 1 — методы влияют.

**Python:**
```python
F, p = stats.f_oneway(method1, method2, method3)
```

## 7. Корреляция Пирсона ($r$)

**Когда:** измерить линейную связь между двумя числами

**Пример:** Зависит ли балл на экзамене от часов подготовки?

Данные: (2ч, 40б), (4ч, 70б), (6ч, 60б), (8ч, 90б), (10ч, 90б)

**Логика:** Если X растёт и Y растёт вместе — числитель большой положительный → r > 0. Если X растёт, Y падает — r < 0. Если нет связи — r ≈ 0.

| Шаг | Что делаем | Действие | Рез. |
|:---:|------------|----------|:----:|
| 1 | $H_0$ | $r = 0$ (связи нет) | |
| | $H_1$ | $r \neq 0$ (связь есть) | |
| 2 | Считаем средние | $\bar{x} = 6$, $\bar{y} = 70$ | |
| 3 | Для каждой точки: отклонение от среднего | $(x-\bar{x})(y-\bar{y})$ | |
| | (2-6)(40-70) = (-4)(-30) = 120 | | |
| | (4-6)(70-70) = (-2)(0) = 0 | | |
| | (6-6)(60-70) = (0)(-10) = 0 | | |
| | (8-6)(90-70) = (2)(20) = 40 | | |
| | (10-6)(90-70) = (4)(20) = 80 | | |
| 4 | Сумма произведений | $120 + 0 + 0 + 40 + 80$ | **240** |
| 5 | Знаменатель | $\sqrt{\sum (x-\bar{x})^2 \cdot \sum (y-\bar{y})^2} = \sqrt{40 \cdot 1800}$ | **≈ 268.3** |
| 6 | **r** | $r = \frac{240}{268.3}$ | **≈ 0.895** |
| 7 | Проверка значимости | $t = \frac{0.895 \sqrt{5-2}}{\sqrt{1-0.895^2}} \approx 3.48$ | |
| 8 | Крит. t (df=3, α=0.05) | $t_{0.025; 3} \approx 3.182$ | |
| 9 | **Сравниваем** | $3.48 > 3.182$ → значимо | |
| 10 | **Вывод** | Связь сильная положительная (≈0.9) | |

**Как понять результат:** r = 0.895 — очень близко к 1. Чем больше часов, тем выше балл. Связь почти линейная.

**Python:**
```python
r, p = stats.pearsonr(x, y)
```

## 8. Множественная корреляция ($R$)

**Когда:** связь одной переменной с несколькими сразу

**Формула (для двух факторов):**
$$R_{1(2,3)}^2 = \frac{r_{12}^2 + r_{13}^2 - 2 r_{12} r_{13} r_{23}}{1 - r_{23}^2}$$

**Пример:** $r_{12}=0.45$, $r_{13}=0.5$, $r_{23}=0.3$, $n=33$

| Шаг | Действие | Рез. |
|:---:|----------|:----:|
| 1 | $R^2 = \frac{0.45^2 + 0.5^2 - 2(0.45)(0.5)(0.3)}{1 - 0.3^2}$ | **≈ 0.349** |
| 2 | $R = \sqrt{0.349}$ | **≈ 0.591** |
| 3 | Проверка $H_0: R=0$: $F = \frac{0.349/2}{(1-0.349)/30}$ | **≈ 8.04** |
| 4 | $F_{0.05}(2, 30)$ из таблицы | **≈ 3.32** |
| 5 | $8.04 > 3.32$ → значимо | |

## 9. Линейная регрессия (предсказание Y по X)

**Когда:** хотим предсказывать Y по X

**Пример:** Данные (1,3), (2,5), (3,4), (4,7), (5,11). Предскажи Y при X=6.

**Логика:** проводим прямую $\hat{Y} = \theta_0 + \theta_1 X$, чтобы сумма квадратов отклонений точек от прямой была минимальной (МНК).

| Шаг | Что делаем | Действие | Рез. |
|:---:|------------|----------|:----:|
| 1 | Строим таблицу | | |
| | X = [1,2,3,4,5] | Y = [3,5,4,7,11] | |
| 2 | Считаем суммы | $\sum X = 15$, $\sum Y = 30$, $\sum XY = 108$ | |
| | | $\sum X^2 = 55$, $n=5$ | |
| 3 | **Наклон $\theta_1$** | $\theta_1 = \frac{n\sum XY - \sum X \sum Y}{n\sum X^2 - (\sum X)^2}$ | |
| | | $= \frac{5\cdot108 - 15\cdot30}{5\cdot55 - 225} = \frac{540 - 450}{275 - 225}$ | **= 1.8** |
| 4 | **Сдвиг $\theta_0$** | $\theta_0 = \bar{Y} - \theta_1 \bar{X} = 6 - 1.8\cdot 3$ | **= 0.6** |
| 5 | **Уравнение** | $\hat{Y} = 0.6 + 1.8X$ | |
| 6 | Проверка наклона: | | |
| | SSE (сумма ошибок) | $\sum (Y - \hat{Y})^2 = 7.6$ | |
| | $s^2 = SSE/(n-2)$ | $= 7.6/3$ | **≈ 2.533** |
| | $SE(\theta_1) = \sqrt{\frac{s^2}{\sum (X-\bar{X})^2}}$ | $= \sqrt{2.533/10}$ | **≈ 0.503** |
| | $t = \theta_1 / SE(\theta_1)$ | $= 1.8 / 0.503$ | **≈ 3.576** |
| | $t_{кр}(0.025; 3)$ | из таблицы Стьюдента | **≈ 3.182** |
| | $3.576 > 3.182$ → наклон значим ✓ | | |
| 7 | **Предсказание** | $\hat{Y}(6) = 0.6 + 1.8 \cdot 6 = 0.6 + 10.8$ | **= 11.4** |

**Как читать:** при X=6 (часов подготовки) ожидаемый Y ≈ 11.4 (баллов). Каждый час подготовки добавляет 1.8 балла.

**Python:**
```python
slope, intercept, r, p, se = stats.linregress(x, y)
# slope=1.8, intercept=0.6, r=0.90, p≈0.037
```

## 10. Спирмен ($\rho$) — ранговый коэффициент

**Когда:** данные не нормальные или связь нелинейная

**Пример:** 6 смартфонов. Цена и качество. Связь?

| Шаг | Что делаем | Действие | Рез. |
|:---:|------------|----------|:----:|
| 1 | Сортируем X по возрастанию, даём ранги | 20→1, 30→2, 35→3, 45→4, 50→5, 80→6 | |
| 2 | То же для Y | 40→1, 50→2, 60→3, 75→4, 80→5, 90→6 | |
| 3 | Считаем $d = R_x - R_y$ | 0, -1, 1, 0, 0, 0 | |
| 4 | $d^2$ | 0, 1, 1, 0, 0, 0 | |
| 5 | $\sum d^2$ | | **2** |
| 6 | **Статистика** | $r_s = 1 - \frac{6 \cdot 2}{6(36-1)} = 1 - \frac{12}{210}$ | **≈ 0.943** |
| 7 | Крит. значение (n=6, α=0.05) | Из таблицы Спирмена | **0.829** |
| 8 | **Сравниваем** | $0.943 > 0.829$? Да ✓ | |
| 9 | **Вывод** | Сильная положительная связь | |

**Почему не Пирсон?** Если данных мало (n=6) или есть выбросы, Спирмен надёжнее — он смотрит не на числа, а на порядок (кто выше, кто ниже).

**Python:**
```python
rho, p = stats.spearmanr(x, y)
```

## 11. Кендалл ($\tau$) — альтернатива Спирмену

**Когда:** много одинаковых значений (связей) или нужна более робастная мера

**Пример:** 5 недель. Затраты на рекламу и число клиентов.

| Шаг | Что делаем | Действие | Рез. |
|:---:|------------|----------|:----:|
| 1 | Берём все пары точек | Всего $C_5^2 = 10$ пар | **10** |
| 2 | Для каждой пары: X и Y растут в одну сторону? | | |
| | (1,2): X↑ Y↑ → согласная | C=1 | |
| | (1,3): X↑ Y↑ → согласная | C=2 | |
| | (1,4): X↑ Y↑ → согласная | C=3 | |
| | (1,5): X↑ Y↑ → согласная | C=4 | |
| | (2,3): X= Y= → ничья | | |
| | (2,4): X↑ Y= → ничья | | |
| | (2,5): X↑ Y↑ → согласная | C=5 | |
| | (3,4): X↑ Y= → ничья | | |
| | (3,5): X↑ Y↑ → согласная | C=6 | |
| | (4,5): X↑ Y↑ → согласная | C=7 | |
| | Всего согласных: $C$ = | | **9** |
| | Несогласных: $D$ = | | **1** |
| 3 | **Статистика** | $\tau = \frac{9-1}{10}$ | **0.8** |
| 4 | **Вывод** | Сильная положительная связь | |

**Чем от Спирмена отличается?** τ Кендалла обычно меньше по модулю, но более устойчив к выбросам и проще интерпретируется как «вероятность согласованности минус несогласованности».

**Python:**
```python
tau, p = stats.kendalltau(x, y)
```

## 12. Вилкоксона (Манн-Уитни) — две группы

**Когда:** ненормальные данные, сравнить две группы

**Пример:** Две выборки по 11 значений. Одинаковы ли?

| Шаг | Что делаем | Действие | Рез. |
|:---:|------------|----------|:----:|
| 1 | $H_0$ | Распределения одинаковы | |
| 2 | Объединяем обе выборки, сортируем | | |
| 3 | Считаем $AB$ — сколько раз X > Y | | **58** |
| 4 | Критические границы (n1=n2=11, α=0.05) | Из таблицы | **[52; 79]** |
| 5 | **Сравниваем** | $58 \in [52; 79]$? Да (внутри) | |
| 6 | **Вывод** | $H_0$ **не отвергаем** | |

**Логика:** Если распределения одинаковы, то значения из обеих групп будут перемешаны случайно, и AB будет около среднего. Если AB выходит за границы — группы различаются.

**Python:**
```python
stat, p = stats.mannwhitneyu(x, y)
```

## 13. Колмогоров-Смирнов (два распределения)

**Когда:** сравнить всю форму распределения, а не только среднее

**Пример:** Север (7 городов) и Юг (7 городов). Одинаково ли распределены температуры?

| Шаг | Что делаем | Действие | Рез. |
|:---:|------------|----------|:----:|
| 1 | Сортируем обе выборки | Север: 15,18,21,24,27,30,33 | |
| | | Юг: 25,28,31,34,37,40,43 | |
| 2 | Строим $F_n(x)$ — накопленные доли | Шаг = $1/n$ | |
| 3 | Находим **максимальное** расхождение | $D = \max |F_{север}(x) - F_{юг}(x)|$ | **0.857** |
| 4 | Крит. значение | $D_{кр} = 1.36 \sqrt{\frac{7+7}{7\cdot7}} \approx 0.728$ | **0.728** |
| 5 | **Сравниваем** | $0.857 > 0.728$? Да ✓ | |
| 6 | **Вывод** | Распределения **разные** | |

**Логика:** Строим два графика накопленных частот. Если они далеко друг от друга — распределения разные. Если близко — одинаковые.

**Python:**
```python
stat, p = stats.ks_2samp(north, south)
```

## 14. Краскел-Уоллиса (3+ группы, ненормальные)

**Когда:** нужно сравнить 3+ группы, данные ненормальные

**Пример:** Три группы студентов. Разные методы обучения. Оценки:

Группа A: [средние ранги → 50 сумма]
Группа B: [сумма рангов 93]
Группа C: [сумма рангов 28]

| Шаг | Что делаем | Действие | Рез. |
|:---:|------------|----------|:----:|
| 1 | Объединяем все группы, сортируем | | |
| 2 | Даём ранги (каждому значению — номер по порядку) | | |
| 3 | Суммируем ранги в каждой группе | $R_1=50$, $R_2=93$, $R_3=28$ | |
| 4 | **Статистика H** | $H = \frac{12}{N(N+1)} \sum \frac{R_i^2}{n_i} - 3(N+1)$ | **≈ 12.93** |
| 5 | p-value | по χ² распределению (df = k-1 = 2) | **≈ 0.0016** |
| 6 | **Сравниваем** | $0.0016 < 0.05$? Да ✓ | |
| 7 | **Вывод** | Группы различаются | |

**Логика:** Если группы одинаковы, ранги распределятся равномерно. Если одна группа систематически получает более высокие ранги — она отличается.

**Python:**
```python
H, p = stats.kruskal(group1, group2, group3)
```

## 15. Джонкхир-Терпстра (проверка тренда)

**Когда:** группы упорядочены (1-й курс < 2-й < 3-й) и ожидаем тренд

**Пример:** Время сборки детали у новичков, 1-3 года опыта, профи. Ожидаем, что с опытом время падает.

| Шаг | Что делаем | Действие | Рез. |
|:---:|------------|----------|:----:|
| 1 | Упорядочиваем группы | Новички < 1-3 года < Профи | |
| 2 | Для каждой пары «ранняя группа — поздняя» считаем, есть ли тренд | | |
| 3 | Всего возможных пар сравнений | 75 пар | **75** |
| 4 | Из них тренд подтверждается | | **71** |
| 5 | **Статистика JT** | 71 из 75 | **71** |
| 6 | p-value | | **<< 0.001** |
| 7 | **Вывод** | Тренд значим — с опытом время сборки падает | |

**Логика:** Если тренда нет, то примерно в половине пар будет «туда», в половине — «обратно». Если почти все пары идут в одну сторону — тренд есть.

## 16. λ и τ Гудмана-Краскала — сила связи для таблиц

**Когда:** после χ² независимости хотим понять, **насколько** сильна связь

**λ (лямбда) — насколько знание X помогает угадать Y:**

$$\hat{\lambda}_B = \frac{\sum_i \max_j n_{ij} - \max_j n_{\cdot j}}{n - \max_j n_{\cdot j}}$$

**Шаги:**
1. Берём таблицу
2. Считаем максимум в каждой строке → суммируем
3. Вычитаем максимум суммы столбцов
4. Делим на n минус максимум суммы столбцов

**Пример:** λ ≈ 0.037 = знание транспорта улучшает предсказание района на 3.7% (слабо)

**τ (тау) — то же, но точнее (учитывает все категории):**

$$\hat{\tau}_B = \frac{n \sum \frac{n_{ij}^2}{n_{i\cdot}} - \sum n_{\cdot j}^2}{n^2 - \sum n_{\cdot j}^2}$$

**Пример:** τ ≈ 0.073 = улучшение на 7.3%

## ВСПОМОГАТЕЛЬНЫЕ ТЕМЫ

### Как построить вариационный ряд

| Шаг | Действие | Пример |
|:---:|----------|--------|
| 1 | Собрать данные | Рост 112 студентов |
| 2 | Найти мин и макс | 152 и 197 |
| 3 | Размах = макс - мин | 45 |
| 4 | Выбрать число интервалов $k$ | По формуле Стерджесса или из условия |
| 5 | Длина интервала $h = (макс - мин)/k$ | 45/9 = 5 |
| 6 | Разбить на интервалы | 150-155, 155-160, ... |
| 7 | Посчитать частоту каждого | Сколько студентов в каждом |

### Как посчитать ряд распределения

1. Перечислить все возможные исходы (k = 0, 1, 2, 3)
2. Посчитать вероятность каждого
3. $E(\xi) = \sum k \cdot P(k)$ — среднее
4. $E(\xi^2) = \sum k^2 \cdot P(k)$
5. $D(\xi) = E(\xi^2) - (E(\xi))^2$ — разброс

**Пример:** три студента сдают экзамен. Вероятности: 0.8, 0.7, 0.9.
- P(0 сдал) = 0.2 × 0.3 × 0.1 = 0.006
- P(1 сдал) = 0.8×0.3×0.1 + 0.2×0.7×0.1 + 0.2×0.3×0.9 = 0.092
- P(2 сдал) = 0.8×0.7×0.1 + 0.8×0.3×0.9 + 0.2×0.7×0.9 = 0.398
- P(3 сдал) = 0.8×0.7×0.9 = 0.504
- E = 0×0.006 + 1×0.092 + 2×0.398 + 3×0.504 = **2.4**
- D = (0²×0.006 + 1²×0.092 + 4×0.398 + 9×0.504) - 2.4² = 6.22 - 5.76 = **0.46**

### Несмещённость и робастность

**Несмещённость:** если брать много разных выборок и усреднять оценки — попадём в истинное значение.
- $\bar{x}$ — несмещённая оценка для $\lambda$ Пуассона ($E[\bar{X}] = \lambda$)

**Робастность:** оценка не ломается от выбросов.
- **Среднее** — НЕ робастно: [1,2,3,4,100] → среднее 22
- **Медиана** — робастна: [1,2,3,4,100] → медиана 3
- **Усечённое среднее** — робастно: отбросили крайние 20%, осталось (2+3+4)/3 = 3

## КАК ВЫБИРАТЬ КРИТЕРИЙ (алгоритм)

```
Сколько групп?
│
├─ ОДНА группа:
│  ├─ Сравнить среднее с числом?
│  │  └─ t-тест (или z, если σ известна)
│  └─ Проверить форму распределения?
│     └─ χ² согласия или K-S
│
├─ ДВЕ группы:
│  ├─ Сравнить средние?
│  │  ├─ Данные нормальные → t-тест
│  │  └─ Ненормальные → Вилкоксон (Манн-Уитни)
│  ├─ Сравнить дисперсии?
│  │  └─ F-тест
│  └─ Сравнить распределения целиком?
│     └─ Колмогоров-Смирнов
│
├─ ТРИ+ группы:
│  ├─ Нормальные → ANOVA
│  ├─ Ненормальные → Краскел-Уоллис
│  └─ Есть порядок групп → Джонкхир-Терпстра
│
├─ СВЯЗЬ между переменными:
│  ├─ Обе числа, нормальные → Пирсон (r)
│  ├─ Обе числа, ненормальные → Спирмен (ρ) или Кендалл (τ)
│  └─ Категориальные → χ² + V Крамера / λ / τ
│
└─ ПРЕДСКАЗАНИЕ Y по X:
   ├─ Один X → линейная регрессия
   └─ Несколько X → множественная регрессия
```

**Главное:** считаешь статистику → сравниваешь с критическим значением или смотришь p-value. Если $p < 0.05$ → «обнаружено различие / связь». Если $p \geq 0.05$ → «недостаточно данных, чтобы утверждать».

## PYTHON — всё что нужно

```python
import numpy as np
from scipy import stats

# --- Одна группа ---
stats.ttest_1samp(данные, mu0)           # t-тест

stats.ttest_ind(a, b)                    # t-тест для двух
stats.mannwhitneyu(a, b)                 # Вилкоксон
stats.ks_2samp(a, b)                     # K-S 2 выборки

stats.f_oneway(a, b, c)                  # ANOVA
stats.kruskal(a, b, c)                   # Краскел-Уоллис

# --- Связь ---
stats.pearsonr(x, y)                     # Пирсон
stats.spearmanr(x, y)                    # Спирмен
stats.kendalltau(x, y)                   # Кендалл
stats.chi2_contingency(таблица)          # χ² независимости

# --- Регрессия ---
stats.linregress(x, y)                   # линейная регрессия
```
