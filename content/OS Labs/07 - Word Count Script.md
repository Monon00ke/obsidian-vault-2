## 1. Количество слов, начинающихся на заданную букву

```bash

tr -s ' ' '\n' < text.txt | grep '^a' | wc -l

```

## 2. Буква, первое слово на которую появилось позже всех

Референсное решение:

```bash

tr -s ' ' '\n' |

cut -c 1 |

nl -w1 -s : |

sort -t : -k 2,2 -u |

sort -t : -k 1,1 -n |

tail -1 |

cut -d : -f 2

```

**Логика:**

1. `tr -s ' ' '\n'` — разбивает текст на слова (по одному на строку)

2. `cut -c 1` — берёт первую букву каждого слова

3. `nl -w1 -s :` — нумерует строки

4. `sort -t : -k 2,2 -u` — сортирует по букве, оставляя только уникальные

5. `sort -t : -k 1,1 -n` — сортирует по номеру строки (времени появления)

6. `tail -1` — последняя строка = буква, которая появилась позже всех

7. `cut -d : -f 2` — извлекает саму букву

## 3. Буквы, с которых не начиналось ни одного слова

Референсное решение:

```bash

(

(tr -s ' ' '\n' |

cut -c 1 |

nl -w1 -s : |

sort -t : -k 2,2 -u |

sort -t : -k 1,1 -n

) ;

(echo 0:a0:b0:c0:d0:e0:f0:g0:h0:i0:j0:k0:l0:m0:n0:o0:p0:q0:r0:s0:t0:u0:v0:w0:x0:y0:z | fold -w3)

) |

sort -k 2,2 -t : -u |

grep ^0: |

cut -d : -f 2 |

sort

```

**Улучшенная генерация нулей:**

```bash

paste -d : <(yes 0) <(echo abcdefghijklmnopqrstuvwxyz | fold -w 1) | head -26

```

## 4. Right join через join

```bash

tr -s ' ' '\n' |

cut -c 1 |

nl -w1 -s : |

sort -t : -k 2,2 -u |

sort -t : -k 1,1 -n |

sort -t : -k 2,2 |

join -t : -2 2 -1 2 -a 2 - \

<(echo 0:a0:b0:c0:d0:e0:f0:g0:h0:i0:j0:k0:l0:m0:n0:o0:p0:q0:r0:s0:t0:u0:v0:w0:x0:y0:z | fold -w3)

```

## 5. Python-скрипт с подстановкой `<(...)`

```python

import sys

print(sys.argv)

```

Запуск:

```bash

python3 script.py <(echo hello)

```

В `sys.argv` придёт путь к временному файлу, созданному bash.

## 6. Фильтрация через grep

```bash

tr -s ' ' '\n' | grep '^[a-z]'

```

Оставляет только слова, начинающиеся со строчной латинской буквы.
