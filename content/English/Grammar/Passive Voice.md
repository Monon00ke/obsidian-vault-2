# Passive Voice

## Formula

**Active:** Subject  +  Verb  +  Object
**Passive:** Object  +  be  +  V3  (+  by Subject)

| Tense | Active | Passive | Example (Passive) |
|-------|--------|---------|-------------------|
| **Present Simple** | V1 / V-s | am/is/are + V3 | The room **is cleaned** every day. |
| **Present Continuous** | am/is/are + V-ing | am/is/are + being + V3 | The room **is being cleaned** now. |
| **Present Perfect** | have/has + V3 | have/has + been + V3 | The room **has been cleaned**. |
| **Past Simple** | V2 | was/were + V3 | The room **was cleaned** yesterday. |
| **Past Continuous** | was/were + V-ing | was/were + being + V3 | The room **was being cleaned** when I arrived. |
| **Past Perfect** | had + V3 | had + been + V3 | The room **had been cleaned** before we arrived. |
| **Future Simple** | will + V1 | will + be + V3 | The room **will be cleaned** tomorrow. |
| **Modal verbs** | can/must/should + V1 | can/must/should + be + V3 | The room **must be cleaned**. |

## When to Use Passive

| Use Case | Example |
|----------|---------|
| **Agent is unknown** (неизвестно, кто сделал) | My bike **was stolen**. |
| **Agent is obvious / not important** (очевидно) | The murderer **was arrested** yesterday. |
| **Focus on the action, not the doer** (акцент на действии) | The new hospital **will be opened** next year. |
| **Formal / scientific writing** (формальный стиль) | The experiment **was conducted** in 2020. |
| **Polite / avoiding blame** (вежливость) | The window **was broken**. (instead of "You broke it!") |

## When NOT to use Passive

- With **intransitive verbs** (no object): arrive, sleep, cry, go, come, die
  - He **arrived** late. (NOT He was arrived late)
- When the **agent is the main focus**: Shakespeare **wrote** Hamlet. (better than Hamlet was written by Shakespeare)

## The "by" Agent

| When to use "by" | Example |
|------------------|---------|
| **Important information** | The Mona Lisa **was painted by** Leonardo da Vinci. |
| **Not obvious** | The cake **was eaten by** the dog. |

| When to omit "by" | Example |
|-------------------|---------|
| **General people** (people/they/someone) | English **is spoken** all over the world. |
| **Not important** | The window **was broken**. |

## Personal vs Impersonal Passive

| Type | Construction | Example |
|------|-------------|---------|
| **Impersonal** (безличный) | It + passive + that clause | **It is said that** he is rich. |
| **Personal** (личный) | Subject + passive + to-inf | He **is said to be** rich. |

**Common verbs:** believe, claim, consider, expect, know, report, say, think, understand

| Active | Impersonal Passive | Personal Passive |
|--------|-------------------|-----------------|
| People say he is rich. | **It is said that** he is rich. | He **is said to be** rich. |
| Everyone knows she works hard. | **It is known that** she works hard. | She **is known to work** hard. |
| They believe the suspect fled. | **It is believed that** the suspect fled. | The suspect **is believed to have fled**. |

## Get Passive

> **be + V3** (formal) vs **get + V3** (informal, often negative)

| be passive | get passive |
|------------|-------------|
| He **was fired**. | He **got fired**. |
| She **was married** in June. | She **got married** in June. |
| The window **was broken**. | The window **got broken**. |

## Examples

- **Present:** Coffee **is grown** in Brazil.
- **Past:** The letter **was posted** yesterday.
- **Future:** The project **will be completed** soon.
- **Modal:** Your homework **must be handed in** by Friday.
- **Continuous:** The road **is being repaired**.
- **Perfect:** All tickets **have been sold** out.

## Active → Passive Transformation

| Step | Active | Passive |
|------|--------|---------|
| 1 | Shakespeare wrote Hamlet. | |
| 2 | Object → subject | **Hamlet** |
| 3 | Verb → be + V3 | **was written** |
| 4 | Subject → by + agent | **by Shakespeare** |
| **Result** | | **Hamlet was written by Shakespeare.** |

## Common Mistakes

| Wrong | Correct |
|-------|---------|
| It **was happened** yesterday. (happen — intransitive) | It **happened** yesterday. |
| The window **was broke**. (need V3) | The window **was broken**. |
| The car **is been repaired**. | The car **is being repaired**. |
| She **has been be** promoted. | She **has been promoted**. |
