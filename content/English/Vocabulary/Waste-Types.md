# Waste Types — Vocabulary

| Type | Description | Example |
|------|-------------|---------|
| **rubbish** | general waste | everyday household waste |
| **waste** | from factories | industrial waste |
| **trash** | dry waste | paper, plastic |
| **garbage** | organic/natural waste | banana peels, apple cores |
| **litter** | illegal trash in public places | cans on the street |
| **junk** | old, not used items | junk mail, junk food |
| **debris** | scattered fragments | after building deconstruction |
| **sludge** | semi-solid waste | at landfills |
| **scrap** | waste metal | scrap metal recycling |
| **e-waste** | electronic waste | old phones, computers |

> garbage → compost (organic waste can be composted)
