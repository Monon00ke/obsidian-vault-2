**Работа №3**
**Определение фокусного расстояния линз**

**Цель работы:** определение фокусного расстояния собирающей линзы по линейному увеличению и методом Бесселя; определение фокусного расстояния рассеивающей линзы.

**Приборы и принадлежности:** оптическая скамья, собирающая линза, рассеивающая линза, экран, источник света, линейка.

| Название | Пределы измерения | Цена деления | Погрешность |
|---|---|---|---|
| Линейка | 0–1000 мм | 1 мм | 0.5 мм |

**Основные расчётные формулы:**

Формула тонкой линзы:
$$\frac{1}{a} + \frac{1}{b} = \frac{1}{f} \tag{9.1}$$

**Способ 1** — по расстояниям $a$ и $b$ (собирающая):
$$f = \frac{a \cdot b}{a + b} \tag{9.2}$$

**Способ 2 (Бессель)** — по перемещению (собирающая):
$$f = \frac{L^2 - l^2}{4L} \tag{9.6}$$

**Способ 3** — рассеивающая линза:
$$f = \frac{a \cdot b}{a - b} \tag{9.7}$$

Погрешность ($n = 3$, $t_{0.95}(2) = 4.303$):
$$\sigma = \sqrt{\frac{\sum \Delta f_i^2}{n-1}}, \qquad
\Delta f = t \cdot \frac{\sigma}{\sqrt{n}}, \qquad
\varepsilon = \frac{\Delta f}{\overline{f}} \cdot 100\%$$

---

**Результаты измерений и их обработка:**

### 1 способ. Собирающая линза — по $a$ и $b$

$$f_i = \frac{a_i \cdot b_i}{a_i + b_i}$$

| № | $a$, см | $b$, см | $h$, см | $H$, см | $f_i$, см | $\Delta f_i = f_i - \overline{f}$ | $\Delta f_i^2$ |
|---|---|---|---|---|---|---|---|
| 1 | 79 | 25 | 3.1 | 1.3 | $\frac{79 \cdot 25}{79 + 25} = 18.99$ | $+0.05$ | 0.0025 |
| 2 | 72.5 | 25.8 | 3.1 | 1.2 | $\frac{72.5 \cdot 25.8}{72.5 + 25.8} = 19.03$ | $+0.09$ | 0.0081 |
| 3 | 60 | 27.4 | 3.1 | 1.6 | $\frac{60 \cdot 27.4}{60 + 27.4} = 18.81$ | $-0.13$ | 0.0169 |
| **$\overline{f}$** | | | | | **18.94** | $\sum =$ | **0.0275** |
| **$\sigma$** | | | | | **0.12** | | |
| **$\Delta f$** | | | | | **$\pm 0.30$** | | |
| **$\varepsilon$** | | | | | **1.6%** | | |

$$\overline{f} = \frac{f_1 + f_2 + f_3}{n} = \frac{18.99 + 19.03 + 18.81}{3} = 18.94 \text{ см}$$

$$\sigma = \sqrt{\frac{\sum \Delta f_i^2}{n-1}} = \sqrt{\frac{0.0275}{2}} = 0.12 \text{ см}$$

$$\Delta f = t_{0.95}(2) \cdot \frac{\sigma}{\sqrt{n}} = 4.303 \cdot \frac{0.12}{\sqrt{3}} = 0.30 \text{ см}$$

$$\varepsilon = \frac{\Delta f}{\overline{f}} \cdot 100\% = \frac{0.30}{18.94} \cdot 100\% = 1.6\%$$

**Систематическая погрешность (инструментальная):**

Для $f = \dfrac{a \cdot b}{a + b}$ частные производные:

$$\frac{\partial f}{\partial a} = \frac{b^2}{(a + b)^2}, \qquad
\frac{\partial f}{\partial b} = \frac{a^2}{(a + b)^2}$$

$$\Delta f_{\text{сист}} = \sqrt{\left(\frac{\partial f}{\partial a} \Delta a\right)^2 + \left(\frac{\partial f}{\partial b} \Delta b\right)^2}$$

При $\Delta a = \Delta b = 0.05$ см (половина цены деления линейки), для первой точки ($a = 79$ см, $b = 25$ см):

$$\frac{\partial f}{\partial a} = \frac{25^2}{(79 + 25)^2} = \frac{625}{10816} = 0.058, \quad
\frac{\partial f}{\partial b} = \frac{79^2}{(79 + 25)^2} = \frac{6241}{10816} = 0.577$$

$$\Delta f_{\text{сист}} = \sqrt{(0.058 \cdot 0.05)^2 + (0.577 \cdot 0.05)^2} = \sqrt{0.0000084 + 0.000832} = 0.029 \text{ см}$$

$$\boxed{f_1 = (18.94 \pm 0.30) \text{ см}}$$

---

### 2 способ (Бессель). Собирающая линза — по перемещению

$$f_i = \frac{L_i^2 - l_i^2}{4 L_i}$$

| № | $L$, см | $l$, см | $f_i$, см | $\Delta f_i = f_i - \overline{f}$ | $\Delta f_i^2$ |
|---|---|---|---|---|---|
| 1 | 100 | 51 | $\frac{100^2 - 51^2}{4 \cdot 100} = 18.50$ | $-0.15$ | 0.0225 |
| 2 | 90 | 37.5 | $\frac{90^2 - 37.5^2}{4 \cdot 90} = 18.59$ | $-0.06$ | 0.0036 |
| 3 | 80 | 19 | $\frac{80^2 - 19^2}{4 \cdot 80} = 18.87$ | $+0.22$ | 0.0484 |
| **$\overline{f}$** | | | **18.65** | $\sum =$ | **0.0745** |
| **$\sigma$** | | | **0.19** | | |
| **$\Delta f$** | | | **$\pm 0.47$** | | |
| **$\varepsilon$** | | | **2.5%** | | |

$$\overline{f} = \frac{f_1 + f_2 + f_3}{n} = \frac{18.50 + 18.59 + 18.87}{3} = 18.65 \text{ см}$$

$$\sigma = \sqrt{\frac{\sum \Delta f_i^2}{n-1}} = \sqrt{\frac{0.0745}{2}} = 0.19 \text{ см}$$

$$\Delta f = t_{0.95}(2) \cdot \frac{\sigma}{\sqrt{n}} = 4.303 \cdot \frac{0.19}{\sqrt{3}} = 0.47 \text{ см}$$

$$\varepsilon = \frac{\Delta f}{\overline{f}} \cdot 100\% = \frac{0.47}{18.65} \cdot 100\% = 2.5\%$$

**Систематическая погрешность (инструментальная):**

Для $f = \dfrac{L^2 - l^2}{4L}$ частные производные:

$$\frac{\partial f}{\partial L} = \frac{L^2 + l^2}{4 L^2}, \qquad
\frac{\partial f}{\partial l} = -\frac{l}{2 L}$$

$$\Delta f_{\text{сист}} = \sqrt{\left(\frac{\partial f}{\partial L} \Delta L\right)^2 + \left(\frac{\partial f}{\partial l} \Delta l\right)^2}$$

При $\Delta L = \Delta l = 0.05$ см, для первой точки ($L = 100$ см, $l = 51$ см):

$$\frac{\partial f}{\partial L} = \frac{100^2 + 51^2}{4 \cdot 100^2} = \frac{10000 + 2601}{40000} = 0.315, \quad
\frac{\partial f}{\partial l} = -\frac{51}{2 \cdot 100} = -0.255$$

$$\Delta f_{\text{сист}} = \sqrt{(0.315 \cdot 0.05)^2 + (0.255 \cdot 0.05)^2} = \sqrt{0.000248 + 0.000163} = 0.020 \text{ см}$$

Оба метода дают согласованные результаты: $18.94$ и $18.65$ см — разница в пределах погрешности.

---

### 3 способ. Рассеивающая линза

$$f_i = \frac{a_i \cdot b_i}{a_i - b_i}$$

| № | $a$, см | $b$, см | $f_i$, см | $\Delta f_i = f_i - \overline{f}$ | $\Delta f_i^2$ |
|---|---|---|---|---|---|
| 1 | 11.7 | 20 | $\frac{11.7 \cdot 20}{11.7 - 20} = -28.19$ | $-2.12$ | 4.49 |
| 2 | 15.5 | 43 | $\frac{15.5 \cdot 43}{15.5 - 43} = -24.24$ | $+1.83$ | 3.35 |
| 3 | 18.5 | 65.5 | $\frac{18.5 \cdot 65.5}{18.5 - 65.5} = -25.78$ | $+0.29$ | 0.08 |
| **$\overline{f}$** | | | **$-26.07$** | $\sum =$ | **7.92** |
| **$\sigma$** | | | **1.99** | | |
| **$\Delta f$** | | | **$\pm 5.0$** | | |
| **$\varepsilon$** | | | **19%** | | |

$$\overline{f} = \frac{f_1 + f_2 + f_3}{n} = \frac{-28.19 - 24.24 - 25.78}{3} = -26.07 \text{ см}$$

$$\sigma = \sqrt{\frac{\sum \Delta f_i^2}{n-1}} = \sqrt{\frac{7.92}{2}} = 1.99 \text{ см}$$

$$\Delta f = t_{0.95}(2) \cdot \frac{\sigma}{\sqrt{n}} = 4.303 \cdot \frac{1.99}{\sqrt{3}} = 5.0 \text{ см}$$

$$\varepsilon = \frac{\Delta f}{|\overline{f}|} \cdot 100\% = \frac{5.0}{26.07} \cdot 100\% = 19\%$$

**Систематическая погрешность (инструментальная):**

Для $f = \dfrac{a \cdot b}{a - b}$ частные производные:

$$\frac{\partial f}{\partial a} = -\frac{b^2}{(a - b)^2}, \qquad
\frac{\partial f}{\partial b} = \frac{a^2}{(a - b)^2}$$

$$\Delta f_{\text{сист}} = \sqrt{\left(\frac{\partial f}{\partial a} \Delta a\right)^2 + \left(\frac{\partial f}{\partial b} \Delta b\right)^2}$$

При $\Delta a = \Delta b = 0.05$ см, для первой точки ($a = 11.7$ см, $b = 20$ см):

$$\frac{\partial f}{\partial a} = -\frac{20^2}{(11.7 - 20)^2} = -\frac{400}{68.89} = -5.81, \quad
\frac{\partial f}{\partial b} = \frac{11.7^2}{(11.7 - 20)^2} = \frac{136.9}{68.89} = 1.99$$

$$\Delta f_{\text{сист}} = \sqrt{(5.81 \cdot 0.05)^2 + (1.99 \cdot 0.05)^2} = \sqrt{0.0844 + 0.0099} = 0.31 \text{ см}$$

$$\boxed{f_3 = (-26.1 \pm 5.0) \text{ см}}$$

---

**Вывод:**

1. Собирающая линза:
   - способ 1 (по $a$ и $b$): $f_1 = (18.94 \pm 0.30)$ см
   - способ Бесселя: $f_2 = (18.65 \pm 0.47)$ см
   - результаты согласованы.

2. Рассеивающая линза:
   - $f_3 = (-26.1 \pm 5.0)$ см
   - знак минус соответствует рассеивающей линзе.

3. Большая погрешность рассеивающей линзы ($19\%$) обусловлена малыми $a$ и большим разбросом $b$.
