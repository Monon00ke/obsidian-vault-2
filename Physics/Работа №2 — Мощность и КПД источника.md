**Исследование зависимости полной мощности, полезной мощности и КПД источника от нагрузки**

**Цель работы:** изучение зависимостей полной мощности $P$, полезной мощности $P_n$ и КПД $\eta$ источника от тока (сопротивления) нагрузки; определение ЭДС $\mathcal{E}$ и внутреннего сопротивления $r$ источника.

**Приборы и принадлежности:** источник питания, вольтметр, амперметр, магазин сопротивлений, ключ, соединительные провода.

| Название | Пределы измерения | Цена деления | Погрешность |
|---|---|---|---|
| Вольтметр | 0–2.5 В | 0.02 В | 0.015 В |
| Амперметр | 0–0.75 А | 0.005 А | 0.0015 А |
| Магазин сопротивлений | 0–9999 Ом | 1 Ом | 0.5 Ом |

**Основные расчётные формулы:**

Полная мощность:
$$P = \mathcal{E} I \tag{2.1}$$

Напряжение на нагрузке:
$$U = \mathcal{E} - I r \tag{2.2}$$

Полезная мощность:
$$P_n = I U = \mathcal{E} I - I^2 r \tag{2.3}$$

Коэффициент полезного действия:
$$\eta = \frac{P_n}{P} = \frac{U}{\mathcal{E}} = \frac{R}{R + r} \tag{2.4}$$

Максимум $P_n$ достигается при $R = r$ (согласованная нагрузка), тогда:
$$I = \frac{\mathcal{E}}{2r}, \quad P_{n,\max} = \frac{\mathcal{E}^2}{4r}, \quad \eta = 50\%$$

**1. Определение $\mathcal{E}$ и $r$ методом наименьших квадратов**

Зависимость $U = \mathcal{E} - r I$ — линейная. Введём обозначения:
$$y = U, \quad x = I, \quad a = \mathcal{E}, \quad b = -r$$

Уравнение регрессии: $y = a + b x$.

Коэффициенты МНК:

$$b = \frac{n \sum x_i y_i - \sum x_i \sum y_i}{n \sum x_i^2 - (\sum x_i)^2}, \qquad
a = \frac{\sum y_i - b \sum x_i}{n}$$

Вычисляем суммы:

$\sum I = 0.246 + 0.228 + 0.210 + 0.194 + 0.180 + 0.160 + 0.154 + 0.144 + 0.128 + 0.116$  
$\qquad + 0.100 + 0.084 + 0.070 + 0.054 + 0.026 + 0.018 + 0.002 = 2.114$ А

$\sum U = 0.10 + 0.30 + 0.448 + 0.64 + 0.76 + 0.86 + 0.96 + 1.02 + 1.16 + 1.26$  
$\qquad + 1.36 + 1.44 + 1.58 + 1.70 + 1.84 + 1.98 + 2.08 = 19.488$ В

$\sum IU = 24.6 + 68.4 + 94.1 + 124.2 + 136.8 + 137.6 + 147.8 + 146.9 + 148.5 + 146.2$  
$\qquad + 136.0 + 121.0 + 110.6 + 91.8 + 47.8 + 35.6 + 4.2 = 1722.1 \cdot 10^{-3} = 1.7221$

$\sum I^2 = (60.52 + 51.98 + 44.10 + 37.64 + 32.40 + 25.60 + 23.72 + 20.74 + 16.38 + 13.46$  
$\qquad + 10.00 + 7.06 + 4.90 + 2.92 + 0.68 + 0.32 + 0.004) \cdot 10^{-3} = 0.3524$

$n = 17$

Подставляем:

$$b = \frac{n \sum I_i U_i - \sum I_i \sum U_i}{n \sum I_i^2 - (\sum I_i)^2}
= \frac{17 \cdot 1.7220 - 2.114 \cdot 19.488}{17 \cdot 0.3524 - 2.114^2}
= \frac{29.274 - 41.198}{5.991 - 4.469}
= \frac{-11.924}{1.522}
= -7.83$$

$$a = \frac{\sum U_i - b \sum I_i}{n}
= \frac{19.488 - (-7.83) \cdot 2.114}{17}
= \frac{19.488 + 16.55}{17}
= \frac{36.04}{17}
= 2.12$$

Следовательно:

$$\mathcal{E} = a = 2.12 \text{ В}, \qquad r = -b = 7.83 \text{ Ом}$$

$$S_{\text{ост}}^2 = \frac{1}{n - 2} \sum \delta_i^2 = \frac{0.0293}{15} = 0.00195 \text{ В}^2$$

Стандартные ошибки коэффициентов регрессии:

$$D = n \sum I_i^2 - (\sum I_i)^2 = 17 \cdot 0.3524 - 2.114^2 = 5.991 - 4.469 = 1.522$$

$$\sigma_b = S_{\text{ост}} \sqrt{\frac{n}{D}} = 0.0442 \sqrt{\frac{17}{1.522}} = 0.0442 \cdot 3.342 = 0.148$$

$$t_{0.95}(15) = 2.131$$

Доверительные интервалы:

$$\Delta \mathcal{E} = \Delta a = t \cdot \sigma_a = 2.131 \cdot 0.0213 \approx 0.045 \text{ В}$$

$$\Delta r = \Delta(-b) = t \cdot \sigma_b = 2.131 \cdot 0.148 \approx 0.315 \text{ Ом}$$

**3. Определение максимальной полезной мощности**

Из таблицы: максимум $P_n$ наблюдается при $I = 144$ мА, $U = 1.02$ В:

$$P_{n,\max} = 0.138 \text{ Вт}$$

Проверка условия $R = r$ (согласованная нагрузка):

$$R = \frac{U}{I} = \frac{1.02}{0.144} = 7.08 \text{ Ом}$$

$$r = 7.83 \pm 0.32 \text{ Ом}$$

$R = 7.08$ Ом — в пределах $2\sigma$ от $r$. Равенство $R \approx r$ выполняется.

КПД при максимуме $P_n$:

$$\eta = \frac{U}{\mathcal{E}} = \frac{1.02}{2.12} = 0.481 = 48.1\% \approx 50\%$$

Теоретические расчёты для режима $R = r$:

$$I_{\text{теор}} = \frac{\mathcal{E}}{2r} = \frac{2.12}{2 \cdot 7.83} = 0.135 \text{ А} = 135 \text{ мА}$$

$$P_{n,\max,\text{теор}} = \frac{\mathcal{E}^2}{4r} = \frac{2.12^2}{4 \cdot 7.83} = \frac{4.49}{31.32} = 0.143 \text{ Вт}$$

Экспериментальные значения ($I = 144$ мА, $P_n = 0.138$ Вт, $\eta = 48\%$) согласуются с теорией ($I = 135$ мА, $P_n = 0.143$ Вт, $\eta = 50\%$) в пределах погрешности.

Ток короткого замыкания:

$$I_{\max} = \frac{\mathcal{E}}{r} = \frac{2.12}{7.83} = 0.271 \text{ А} = 271 \text{ мА}$$

**4. Погрешности ($P_n$, $\eta$)**

Инструментальная погрешность вольтметра: $\Delta U = 0.015$ В  
Инструментальная погрешность амперметра: $\Delta I = 0.0015$ А  
Погрешность МНК: $\Delta \mathcal{E} = 0.05$ В

Погрешность косвенных измерений (на примере точки №8, $I = 0.144$ А, $U = 1.02$ В):

$$\varepsilon_{P_n} = \frac{\Delta P_n}{P_n} = \sqrt{\left(\frac{\Delta I}{I}\right)^2 + \left(\frac{\Delta U}{U}\right)^2}
= \sqrt{\left(\frac{0.0015}{0.144}\right)^2 + \left(\frac{0.015}{1.02}\right)^2}
= \sqrt{0.0104^2 + 0.0147^2}
= 1.8\%$$

$$\varepsilon_{\eta} = \frac{\Delta \eta}{\eta} = \sqrt{\left(\frac{\Delta U}{U}\right)^2 + \left(\frac{\Delta \mathcal{E}}{\mathcal{E}}\right)^2}
= \sqrt{\left(\frac{0.015}{1.02}\right)^2 + \left(\frac{0.05}{2.12}\right)^2}
= \sqrt{0.0147^2 + 0.0236^2}
= 2.8\%$$
