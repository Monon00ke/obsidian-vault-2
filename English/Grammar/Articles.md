# Articles (a / an / the / Zero)

## The Basics

| Article | When to Use | Example |
|---------|-------------|---------|
| **a** | before consonant sound | a book, a university, a European |
| **an** | before vowel sound | an apple, an hour, an honest man |
| **the** | specific / known | the book that I bought |
| **Zero** (∅) | general / abstract | I like ∅ music. |

## A / An (Indefinite Article) — первый раз, неопределённый

| Use Case | Example |
|----------|---------|
| **First mention** (первое упоминание) | I saw **a** dog in the park. |
| **One of many / any** (один из многих) | She is **a** teacher. |
| **With professions** (профессии) | He is **an** engineer. |
| **With descriptions** (описания) | It was **a** beautiful day. |
| **Price / speed / frequency** (цена, скорость) | 50 km **an** hour, twice **a** month |
| **Exclamations with countable nouns** | What **a** nice day! Such **a** shame! |

> **a/an** = only with **countable singular nouns**

## The (Definite Article) — уже известно / единственный

| Use Case | Example |
|----------|---------|
| **Already mentioned** (уже упомянуто) | I saw a dog. **The** dog was black. |
| **Unique things** (единственные в мире) | **the** sun, **the** Moon, **the** Earth |
| **Specific / known from context** (понятно из контекста) | Close **the** door. (we know which one) |
| **Superlatives** (превосходная степень) | **the** best, **the** most expensive |
| **Ordinal numbers** (порядковые) | **the** first, **the** second, **the** last |
| **Musical instruments** (музыкальные инструменты) | play **the** guitar, **the** piano |
| **Groups of people** (группы людей) | **the** rich, **the** poor, **the** elderly |
| **Nationalities as groups** (национальности) | **the** British, **the** French |
| **Adjectives as nouns** | **the** unemployed, **the** disabled |
| **Parts of something** (части целого) | **the** beginning, **the** end, **the** middle |
| **Countries with plural / keywords** | **the** USA, **the** UK, **the** Netherlands |
| **Rivers, seas, oceans, deserts, mountain ranges** | **the** Volga, **the** Black Sea, **the** Alps |

## Zero Article (∅) — обобщение / абстракция

| Use Case | Example |
|----------|---------|
| **General plural** (обобщение, множ. число) | I love ∅ dogs. (all dogs in general) |
| **Abstract uncountable** (абстрактные) | ∅ Love is beautiful. ∅ Music is art. |
| **Names of people** (имена) | ∅ John, ∅ Mary |
| **Most country names** | ∅ Russia, ∅ France, ∅ Germany |
| **Cities, streets, parks** | ∅ Moscow, ∅ Oxford Street, ∅ Hyde Park |
| **Languages and academic subjects** | I speak ∅ English. ∅ Math is hard. |
| **Meals (general)** | after ∅ breakfast, at ∅ lunch |
| **Sports** | play ∅ football, play ∅ chess |
| **Colours** | I like ∅ red. |
| **Titles when used as name** | ∅ Professor Smith, ∅ Doctor Brown |
| **Shops, companies, airports** | go to ∅ work, at ∅ home, to ∅ bed |
| **Transport expressions** | by ∅ car, by ∅ bus, on ∅ foot |

## Key Rules Comparison

| Situation | a/an | the | ∅ (zero) |
|-----------|------|-----|---------|
| **Countable singular** | I need **a** pen. | **The** pen is mine. | — |
| **Countable plural** | — | **The** pens are on the table. | ∅ Pens are useful. |
| **Uncountable** | — | **The** water is cold. | ∅ Water is essential. |
| **First mention** | I saw **a** cat. | — | — |
| **Second mention** | — | **The** cat was cute. | — |

## Common Fixed Expressions

### With "the"
| Expression | Example |
|------------|---------|
| in **the** morning / afternoon / evening | I work in **the** morning. |
| on **the** left / right | Turn to **the** left. |
| at **the** top / bottom | Sign at **the** bottom. |
| **the** same | We are **the** same age. |
| **the** only | She is **the** only one. |
| play **the** piano / guitar / violin | She plays **the** piano. |
| go to **the** cinema / theatre | Let's go to **the** cinema. |
| listen to **the** radio | I heard it on **the** radio. |

### Without "the" (zero)
| Expression | Example |
|------------|---------|
| at ∅ home / ∅ school / ∅ work | I'm at ∅ home. |
| go to ∅ bed / ∅ hospital / ∅ prison | He went to ∅ bed. |
| by ∅ car / ∅ bus / ∅ train / ∅ plane | I travel by ∅ car. |
| on ∅ foot | I go on ∅ foot. |
| have ∅ breakfast / ∅ lunch / ∅ dinner | Let's have ∅ lunch. |
| in ∅ hospital / in ∅ prison (as patient/prisoner) | He is in ∅ hospital. |
| ∅ next week / ∅ last month | See you ∅ next week. |

### Names with "the"
| With the | Without the |
|----------|-------------|
| **the** Netherlands, **the** USA, **the** UK | ∅ Russia, ∅ France, ∅ Germany |
| **the** Sahara, **the** Alps | ∅ Mount Everest, ∅ Lake Baikal |
| **the** Atlantic Ocean, **the** Nile | ∅ North America, ∅ Africa |
| **the** Hilton, **the** Ritz (hotels) | ∅ Buckingham Palace, ∅ Oxford University |
| **the** Times, **the** Guardian (newspapers) | ∅ CNN, ∅ BBC (TV channels: the BBC is also possible) |
| **the** United Nations, **the** EU | ∅ NATO, ∅ Parliament |
| **the** Titanic, **the** Queen Mary (ships) | ∅ Star Wars, ∅ Harry Potter (titles) |
| **the** Smiths, **the** Johnsons (family) | ∅ Smith, ∅ Johnson (surname) |

## Common Mistakes

| Wrong | Correct |
|-------|---------|
| I saw **the** dog yesterday. (first mention) | I saw **a** dog yesterday. |
| **The** dogs are animals. (general) | ∅ Dogs are animals. |
| I like **the** music. (general) | I like ∅ music. |
| She plays **the** tennis. (sport — no the) | She plays ∅ tennis. |
| He is **a** best student. (superlative → the) | He is **the** best student. |
| I go to **the** bed. (fixed expression) | I go to ∅ bed. |
| **The** Moscow is beautiful. (cities — no article) | ∅ Moscow is beautiful. |
| I speak **the** English. (languages — no article) | I speak ∅ English. |
| at ∅ night (exception) | at ∅ night. (not the night) |
