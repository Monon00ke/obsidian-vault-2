# Conditionals

## Zero Conditional — General Truths (100% true)

| | Structure |
|---|-----------|
| **Formula** | If + Present Simple, **→** Present Simple |
| **Use** | Facts, general truths, scientific facts, habits |
| **Translation** | Если …, то … (всегда) |

| If clause (condition) | Main clause (result) |
|----------------------|---------------------|
| If you **heat** ice, | it **melts**. |
| If I **am** tired, | I **go** to bed early. |
| If it **rains**, | the grass **gets** wet. |
| If she **doesn't water** the plants, | they **die**. |

**Notes:** можно заменить if на when — смысл не меняется.

---

## First Conditional — Real Future (50–90% likely)

| | Structure |
|---|-----------|
| **Formula** | If + Present Simple, **→** will + V1 |
| **Use** | Real/probable situations in the future |
| **Translation** | Если …, то … (реально, может случиться) |

| If clause (condition) | Main clause (result) |
|----------------------|---------------------|
| If it **rains**, | I **will take** an umbrella. |
| If you **study** hard, | you **will pass** the exam. |
| If she **doesn't hurry**, | she **will miss** the bus. |
| If they **invite** me, | I **will go**. |

| Alternative modals in main clause | Example |
|-----------------------------------|---------|
| **can** (можешь) | If you finish early, you **can** go home. |
| **must** (должен) | If you want to pass, you **must** study. |
| **might** (возможно) | If it's sunny, we **might** go to the beach. |
| **should** (следует) | If you're sick, you **should** stay home. |
| **imperative** (повелительное) | If you need help, **call** me. |

## Second Conditional — Unreal Present/Future (0% or unlikely)

| | Structure |
|---|-----------|
| **Formula** | If + Past Simple, **→** would + V1 |
| **Use** | Unreal / hypothetical / impossible present or future |
| **Translation** | Если бы …, то … (нереально, маловероятно, мечты) |

| If clause (condition) | Main clause (result) |
|----------------------|---------------------|
| If I **had** a million dollars, | I **would travel** the world. |
| If I **were** you, | I **wouldn't do** that. |
| If she **spoke** English better, | she **would get** the job. |
| If it **wasn't** so cold, | we **would go** for a walk. |

| Alternative modals | Example |
|--------------------|---------|
| **could** (мог бы) | If I were rich, I **could buy** a house. |
| **might** (возможно бы) | If he tried harder, he **might succeed**. |

> **Важно:** для I/he/she/it во Second Conditional используем **were** (особенно в формальном английском):
> If I **were** you, If he **were** here, If she **were** older.

**"If I were you"** — устойчивое выражение для советов:
- If I **were** you, I **would** apologise.
- If I **were** you, I **wouldn't** take that job.

## Third Conditional — Unreal Past (regret / missed opportunities)

| | Structure |
|---|-----------|
| **Formula** | If + Past Perfect, **→** would have + V3 |
| **Use** | Unreal past — something didn't happen, we imagine the opposite |
| **Translation** | Если бы …, то … (но в прошлом всё было иначе, сожаление) |

| If clause (condition) | Main clause (result) |
|----------------------|---------------------|
| If I **had studied** harder, | I **would have passed** the exam. |
| If we **had left** earlier, | we **wouldn't have missed** the train. |
| If she **had known**, | she **would have come**. |
| If they **hadn't played** so loud, | the neighbours **wouldn't have complained**. |

**Meaning:** I didn't study → I didn't pass. (both are opposite in reality)

## Mixed Conditionals

### Type 1: Past condition → Present result

| | Structure | Example |
|---|-----------|---------|
| **Formula** | If + Past Perfect (past), **→** would + V1 (present) |
| **Use** | The past action affects the present |

**Example:** If I **had taken** that job (past), I **would be** rich now (present).

### Type 2: General truth → Past result

| | Structure | Example |
|---|-----------|---------|
| **Formula** | If + Past Simple (general), **→** would have + V3 (past) |
| **Use** | A general characteristic affected a specific past action |

**Example:** If he **was** more careful (general trait), he **wouldn't have made** that mistake (past).

## Comparison Table

| Conditional | If clause | Main clause | Probability | Time |
|-------------|-----------|-------------|-------------|------|
| **Zero** | Present Simple | Present Simple | 100% | always |
| **First** | Present Simple | will + V1 | ~50–90% | future |
| **Second** | Past Simple | would + V1 | ~0% | present/future |
| **Third** | Past Perfect | would have + V3 | 0% (past) | past |
| **Mixed** | Past Perfect / Past Simple | would + V1 / would have + V3 | 0% | past→present |

## Inversion (Formal)

Instead of "if", we can invert subject and auxiliary:

| Normal | Inverted (formal) |
|--------|------------------|
| If I **had** known… | **Had** I known… |
| If I **were** you… | **Were** I you… |
| If it **should** rain… | **Should** it rain… |

---

## Unless / Even if / Provided / As long as

| Alternative | Meaning | Example |
|-------------|---------|---------|
| **Unless** = if not | если не | I won't go **unless** you come with me. |
| **Even if** | даже если | **Even if** it rains, we'll still go. |
| **Provided (that)** = if | при условии что | You can go **provided** you finish your homework. |
| **As long as** = if | пока / при условии | You can borrow it **as long as** you return it. |

## Common Mistakes

| Wrong | Correct |
|-------|---------|
| If I **will see** him, I'll tell him. (First: no will in if-clause) | If I **see** him, I'll tell him. |
| If I **would have** money, I'd travel. (Second: no would in if-clause) | If I **had** money, I'd travel. |
| If I **would** know, I **would** come. (Second) | If I **knew**, I **would** come. |
| If I **was** you… (informal OK, formal: were) | If I **were** you… |
| When I will arrive… (time clause, not condition) | When I arrive… |
