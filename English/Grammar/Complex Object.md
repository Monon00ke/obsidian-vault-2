# Complex Object (Сложное дополнение)

## What is Complex Object?

Сложное дополнение — конструкция, где после глагола идёт **объект + инфинитив (или V-ing)**. Переводится как придаточное предложение с «что», «чтобы», «как».

| English | Russian |
|---------|---------|
| I want **him to help** me. | Я хочу, **чтобы он помог** мне. |
| She saw **him cross** the street. | Она видела, **как он перешёл** улицу. |

## Structure

Subject + Verb + Object + Infinitive (to V / V / V-ing)

## Type 1: want / would like / expect / need (желание, просьба)

**Formula:** Verb + Object + **to** + V1

| Verb | Example |
|------|---------|
| **want** (хотеть) | I want **you to wait** here. |
| **would like** (хотел бы) | I would like **you to come** early. |
| **expect** (ожидать) | She expects **him to call**. |
| **need** (нуждаться) | I need **you to help** me. |
| **ask** (просить) | She asked **him to wait**. |
| **tell** (велеть, сказать) | He told **me to leave**. |
| **advise** (советовать) | I advise **you to study** more. |
| **allow** (разрешать) | They allowed **us to stay**. |
| **remind** (напоминать) | Remind **me to buy** milk. |
| **encourage** (поощрять) | My parents encourage **me to try**. |

## Type 2: make / let / have (принуждение, разрешение)

**Formula:** Verb + Object + **V1** (without to!)

| Verb | Meaning | Example |
|------|---------|---------|
| **make** | заставлять | She made **me** **clean** the room. |
| **let** | позволять | Let **them** **go** now. |
| **have** | заставлять / просить | I'll have **my assistant** **send** it. |

> ⚠️ В Passive: make → **be made to** + V1
> - Active: She made **me** **clean** it.
> - Passive: I **was made to** **clean** it.

## Type 3: see / hear / watch / notice / feel (восприятие органами чувств)

**Two possible forms:**

### 3a. Full action (законченное действие)
**Formula:** Verb + Object + **V1**
> See/hear the **whole action** from start to finish.

| Verb | Example |
|------|---------|
| **see** | I saw **him** **cross** the street. (видела, как перешёл — целиком) |
| **hear** | She heard **him** **play** the piano. (слышала всю игру) |
| **watch** | We watched **the sun** **set**. |
| **notice** | I noticed **him** **leave** the room. |
| **feel** | She felt **something** **touch** her shoulder. |

### 3b. Action in progress (в процессе)
**Formula:** Verb + Object + **V-ing**
> See/hear **part of the action** — в процессе.

| Verb | Example |
|------|---------|
| **see** | I saw **him** **crossing** the street. (видела, как переходил — в процессе) |
| **hear** | She heard **someone** **singing**. |
| **watch** | I watched **them** **playing** football. |
| **notice** | He noticed **her** **staring** at him. |
| **feel** | I felt **my heart** **beating** faster. |

### Comparison: V1 vs V-ing

| V1 (complete action) | V-ing (in progress) |
|----------------------|---------------------|
| I saw **him** **cross** the road. (he crossed, I saw it all) | I saw **him** **crossing** the road. (he was in the middle) |
| I heard **her** **play** Beethoven. (the whole piece) | I heard **her** **playing** the piano. (she was playing when I came in) |

## Type 4: want / need + Object + V3 (хотеть, чтобы сделали)

**Formula:** want + Object + **V3** (past participle)

| Verb | Example |
|------|---------|
| **want** | I want **this report** **done** by 5 PM. |
| **need** | I need **my car** **fixed**. |
| **would like** | I'd like **the room** **cleaned**. |

## Type 5: consider / believe / find / think (мнение, оценка)

**Formula:** Verb + Object + **to be** (or adjective/noun)

| Verb | Example |
|------|---------|
| **consider** | I consider **him** **to be** the best candidate. |
| **believe** | She believes **him** **to be** honest. |
| **find** | I find **this book** **interesting**. (adjective after object) |
| **think** | I thought **her** **to be** a doctor. |

## Summary Table

| Type | Verbs | Structure | Example |
|------|-------|-----------|---------|
| **1. Wish/Request** | want, would like, expect, need, ask, tell, advise | V + Object + **to** V | I want **you to go**. |
| **2. Force/Permission** | make, let, have | V + Object + **V1** (bare inf) | She made **me clean** it. |
| **3a. Sense (complete)** | see, hear, watch, notice, feel | V + Object + **V1** | I saw **him cross** the road. |
| **3b. Sense (process)** | see, hear, watch, notice, feel | V + Object + **V-ing** | I saw **him crossing** the road. |
| **4. Want + V3** | want, need, would like | V + Object + **V3** | I need **my car fixed**. |
| **5. Opinion** | consider, believe, find, think | V + Object + **to be / adj** | I consider **him intelligent**. |

## Common Mistakes

| Wrong | Correct |
|-------|---------|
| I want **that he helps** me. (that-clause not typical) | I want **him to help** me. |
| She made **me to wait**. (make + to ×) | She made **me wait**. |
| Let **him to go**. (let + to ×) | Let **him go**. |
| I saw **he crossed** the street. (need object, not subject) | I saw **him cross** the street. |
| I heard **her** **to sing**. (sense verb + to ×) | I heard **her** **sing** / **singing**. |
| I want **that my car is repaired**. | I want **my car repaired**. |
